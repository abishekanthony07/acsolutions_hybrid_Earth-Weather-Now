Über die App

Laden Sie aktuelle Wetterdaten für jeden Ort auf der Erde, einschließlich über 200.000 Städten! Diese Wetterdaten werden aus verschiedenen Quellen durch OpenWeather gesammelt und bereitgestellt.

Dabei kann die Postleitzahl oder der Name des Ortes in der Suche eingegeben werden. Hinweis: Es kann hierbei zu ungenauen Angaben des Wetters kommen, wenn der gesuchte Ort nicht gefunden werden kann und alternativ das Wetter einer naheliegenden Stadt angezeigt wird.

Ergänzend zu der einfachen Suche anhand der PLZ / Name des Ortes, können der StateCode und der Ländercode angegeben werden. Wichtig ist, dass der StateCode durch die OpenWeather-API nur im Land USA unterstützt wird.

Eine mögliche Suche könnte lauten:

  • Ort: "Bern"
  • optional Statecode: " "
  • optional Laendercode: "CH"


    • In dieser App wird durch das Speichern der Wettervorhersagen nur die angezeigten Informationen auf Ihrem Smartphone gesichert. Hierbei handelt es sich um:

  • Datum und Uhrzeit der Suche
  • Ihre Sucheingaben
  • Ihr Kommentar beim Speichern
  • Die Wettervorhersage

    OpenWeather - API

    Die OpenWeather - API sorgt dafür, dass die Wettervorhersage des gesuchten (naheliegenden) Ortes aus verschiedenen Quellen dieser App bereitgestellt werden.
    Näheres über die API kann auf deren Webseite nachgelesen werden.
